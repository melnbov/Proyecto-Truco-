# Juego del Truco en Python
Este proyecto consiste en la implementación del juego del Truco argentino en consola, donde un jugador puede enfrentarse contra la computadora. El sistema simula el reparto de cartas, el desarrollo de rondas y la toma de decisiones básica de la IA.

#Funcionalidades
Juego jugador vs computadora
Reparto automático de cartas
Desarrollo de rondas (manos)
Cálculo de puntaje
Sistema básico de decisiones de la computadora:
  Canta Envido cuando tiene más de 24 puntos
  Canta Truco cuando posee cartas fuertes
Determinación del ganador de cada ronda
